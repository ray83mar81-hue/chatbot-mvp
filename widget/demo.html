Bienvenido a la demo

Esta pagina simula el sitio web de un negocio. El widget de chat aparece en la esquina inferior derecha.

Haz clic en la burbuja azul para abrir el chat y probar preguntas como:

  • Cual es el horario?
  • Donde estan ubicados?
  • Tienen WiFi?
  • Puedo hacer una reserva?
  • Que precios tienen?

Como integrar el widget

Agrega esta linea antes de cerrar </body>:

<script
  src="https://tu-dominio.com/chat-widget.js"
  data-api-url="http://localhost:8000"
  data-business-id="1"
  data-title="Chat"
  data-subtitle="Te responderemos al instante"
  data-primary-color="#2563eb"
  data-welcome="Hola! En que puedo ayudarte?"
></script>
